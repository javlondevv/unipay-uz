## LinkedIn

```
I just open-sourced UniPay UZ — a Python library that unifies Uzbekistan's
payment gateways (Payme, Click, Uzum, Paynet, Octo) behind one consistent API.

Every project here that takes payments re-implements the same glue: different
auth per provider, som-vs-tiyin amount units, and a separate webhook contract
for each. I'd written it three times, so I turned it into a library.

What you get:
• One API across all five gateways
• Webhook base classes for Django, FastAPI and Flask — you write the business
  logic, signature verification and transaction state are handled
• MIT licensed, on PyPI: pip install unipay-uz

If you build for the Uzbek market, I'd genuinely value your feedback on the API
design. Repo (with a Russian README): https://github.com/javlondevv/unipay-uz

#Python #Django #FastAPI #OpenSource #Fintech #Uzbekistan
```
